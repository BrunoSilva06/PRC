Abstract: Para este trabalho foi pedida uma ontologia para o arquivo musical disponibilizado. Em primeira instância foi necessário pensar na ontologia de acordo com a informação presente no dataset, sendo que a lógica da ontologia se encontra disponível no ficheiro "ontologia.txt". Depois da base da ontologia feita, recorreu-se a uma transformação XSLT para povoar a ontologia com as entidades presentes no ficheiro XML, transformação essa presente no ficheiro "transform.xsl". Após a transformação, foi concatenada a informação gerada pela ontologia com a informação gerada pela transformação, dando origem ao ficheiro "musicas.ttl". Após concluída esta fase, foi carregada a ontologia no GraphDB para então se proceder às queries pedidas, queries essas que se encontram no ficheiro "Queries.txt". Os resultados das Queries podem ser encontrados na diretoria "../Results"
